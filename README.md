# Internship Task 2 – Data Cleaning and Preparation

## Objective
To clean and prepare the Global Superstore dataset using Python by handling missing values, removing duplicate records, correcting data types, and exporting a cleaned dataset for further analysis.

## Tools Used
- Python
- Pandas
- NumPy
- GitHub

## Steps Performed
1. Loaded the Global Superstore dataset into Python.
2. Identified and handled missing values.
3. Removed duplicate records.
4. Corrected inconsistent data types.
5. Standardized date formats.
6. Verified data quality and integrity.
7. Exported the cleaned dataset to a new CSV file.

## Results
Successfully prepared a clean and reliable dataset by:
- Removing duplicate entries
- Handling missing values
- Standardizing data formats
- Improving data consistency
- Creating a cleaned dataset ready for analysis

- global_superstore_raw.csv
- global_superstore_cleaned.csv
